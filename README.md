
# React + Flask + Materialui


## Setup

For the backend:

```
pip install -r requirements.txt
```

For the frontend (for Nodejs > 4):

```
npm install -g webpack

npm install -g webpack-dev-server

npm install
```

## Running the app

To directly run on webpack-dev-server

```
npm run debug
```

To run using flask

```
npm run build

python app.py
```
